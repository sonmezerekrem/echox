## v1.1.0

*Released: February 28, 2026*

### Features

- **Logo generation** — New `echox logo [COLOR]` command to generate project logo and favicon SVGs and wire them in `config.json`.
- **GitHub Packages** — Package and workflow configuration for publishing to GitHub Packages (registry, `publishConfig`, and docs in README).

### Improvements

- **Doc cards** — Non-link cards (e.g. “Dark Mode”) get the same hover border as link cards for consistent styling.
- **Styles** — Cleanup of header, tab bar, and prev/next layout (decorative title styles removed, border adjustments).
