## v1.2.0

*Released: February 28, 2026*

### Features

- **Task management** — `tasks.yml` per tab with todo / in-progress / done status. Task list pages, TaskCard component with steps, acceptance criteria, dependencies, and relations. Supports internal doc links and external URLs.
- **Logo generation** — `echox logo [COLOR]` generates `assets/logo.svg` and `assets/favicon.svg` from the first letter of the project name and updates `config.json` with logo and favicon paths.
- **Open source prep** — GPL-3.0-or-later LICENSE, README with install and usage, `.gitignore`, and example `config.json` pointed at the real repo.

### Configuration

- **Publish workflow** — GitHub Actions publish workflow runs only when compilation-related paths change (`bin/**`, `src/**`, `astro.config.mjs`, `package.json`, `package-lock.json`, `.npmrc`).
