## v1.0.0

*Released: February 2026*

The first stable release of Echox (renamed from EchoDocs).

### Features

- **Static site generation** from markdown files using Astro 5
- **Folder-based routing** with automatic tab, group, and page navigation
- **Dark mode** with system preference detection and manual toggle
- **Tailwind color theming** — set `color` in config.json to use any Tailwind palette
- **Syntax highlighting** with Shiki (light and dark themes)
- **Hugeicons integration** — customizable sidebar icons via frontmatter
- **Header actions** — configurable links, buttons, and GitHub icon
- **Search** with Pagefind
- **Responsive layout** with mobile sidebar drawer
- **Plugin system** for remark/rehype extensions
- **Docker support** for containerized builds

### CLI Commands

| Command | Description |
|---------|-------------|
| `echox dev` | Start development server |
| `echox build` | Generate static output to `./dist` |
| `echox preview` | Preview the built site |
| `echox init` | Scaffold a new docs project |
