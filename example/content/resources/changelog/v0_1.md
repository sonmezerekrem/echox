## v0.1.0 — Beta

*Released: November 2025*

Initial beta release for early feedback and testing.

### Features

- Basic markdown rendering with Astro
- Folder-based navigation (tabs + sidebar groups)
- Light theme only
- Minimal styling
- CLI with `dev` and `build` commands

### Known Limitations

- No dark mode support
- No configurable colors
- No icon customization
- Search is not functional
- No plugin system

### Migration to v1.0

If you used the beta, upgrading is straightforward:

1. Update the package: `npm install -g echox@latest`
2. Add a `color` field to `config.json` (optional, defaults to `"blue"`)
3. Add `icon` frontmatter to pages if you want custom sidebar icons
4. Dark mode works automatically — no changes needed
