Callouts highlight important information using blockquote syntax. They support four types: Note, Tip, Warning, and Danger.

## Syntax

Use blockquote with a special prefix:

```markdown
> [!NOTE]
> This is a note.

> [!TIP]
> Here's a helpful tip.

> [!WARNING]
> Be careful with this.

> [!DANGER]
> This action cannot be undone.
```

## Examples

> [!NOTE]
> Echox uses GitHub Flavored Markdown style callouts. The first line must start with `[!TYPE]`.

> [!TIP]
> Use callouts to draw attention to best practices, shortcuts, or helpful hints.

> [!WARNING]
> Warnings are for things that might cause problems if ignored.

> [!DANGER]
> Reserve danger callouts for irreversible or critical actions.
